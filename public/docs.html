1. Introdução

O VALORA é uma aplicação web desenvolvida pela METADAX para cálculo de valuation de empresas e equity pessoal. A ferramenta utiliza múltiplas metodologias de avaliação e gera relatórios técnicos profissionais instantaneamente de forma segura na Vercel.

1.1 Objetivo

Calcular o valuation de empresas de todos os portes, utilizando metodologias reconhecidas e integrando dados de APIs públicas e privadas.

1.2 Escopo

  • Valuation de empresas tradicionais
  • Valuation de empresas de nova economia
  • Valuation de startups
  • Equity pessoal (valuation de profissionais)

2. Metodologias de Valuation

2.1 DCF (Discounted Cash Flow)

O método DCF calcula o valor presente dos fluxos de caixa futuros projetados, descontados a uma taxa que reflete o risco do investimento.

Fórmula: VP = Σ(CFt / (1+r)^t) + VT / (1+r)^n

Onde:

  • CFt = Fluxo de caixa no período t
  • r = Taxa de desconto (WACC estimado em 15%)
  • VT = Valor terminal
  • n = Número de períodos (5 anos)

2.2 Múltiplo de EBITDA

Calcula o valor da empresa multiplicando o EBITDA por um múltiplo específico do setor.

Fórmula: EV = EBITDA × Múltiplo

Os múltiplos variam conforme o setor de atuação da empresa.

2.3 Receita × Fator

Multiplica a receita por um fator específico do setor, ajustado pela taxa de crescimento.

Fórmula: EV = Receita × Fator × Ajuste

2.4 Scorecard Method (Startups)

Método específico para startups que considera mercado endereçável, tração, estágio do produto e investimentos recebidos.

4. Score de Valorização (A-F)

O score de valorização é uma avaliação qualitativa que considera 6 blocos de perguntas:

  1. Desempenho e Eficiência (peso 15%)
  2. Mercado e Estratégia (peso 20%)
  3. Pessoas e Intangíveis (peso 15%)
  4. Capital Interno (peso 20%)
  5. Capital Externo (peso 15%)
  6. Capital de Resultado (peso 15%)

Conversão de Rating

5. APIs Integradas

5.1 BrasilAPI

API pública para consulta de dados de CNPJ. Utilizada para preenchimento automático de informações cadastrais.

Endpoint: GET /api/v1/useapis/cnpj/{cnpj}

5.2 ViaCEP

API pública para consulta de endereços por CEP. Envia `User-Agent` para contornar validações de bot.

Endpoint: GET /api/v1/useapis/cep/{cep}

5.3 AXIO Data (Novidade v1.1)

Hub centralizador de submissões das consultas de CNPJs. Os dados coletados são salvos silenciosamente no endpoint de Data da Metadax utilizando chamadas em back-ground, unificando métricas organizacionais.

Endpoint Oculto: POST /api/v1/useapis/axio/{cnpj}

5.4 APIs METADAX

  • RFN API: Rating de Fontes Negativas
  • System Status: Validação de licença
  • Validate Access: Autenticação de usuário

6. Tipos de Relatório

6.1 Relatório Pessoal

Contém apenas informações do sócio/profissional, focado em Equity Pessoal.

6.2 Relatório Empresarial

Contém dados da empresa e impacto do sócio, sem incluir informações específicas do sócio nem o Rating RFN.

6.3 Relatório Completo

Relatório completo com informações da empresa, do sócio e Rating RFN.
